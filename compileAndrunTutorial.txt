How to compile and run: 

gcc udp_client.c battle_setup.c game_logic.c -o client.exe -lws2_32

gcc udp_server.c -o server.exe -lws2_32


gcc udp_host.c -o host.exe -lws2_32

gcc udp_joiner.c -o joiner.exe -lws2_32

gcc udp_host1.c -o host.exe -lws2_32

gcc udp_joiner1.c -o joiner.exe -lws2_32

gcc udp_host1.c BattleManager.c -o host.exe -lws2_32
gcc udp_joiner1.c BattleManager.c -o joiner.exe -lws2_32

Then run in this order:

> host


> joiner (either spectator or joiner)


> joiner (joiner or spectator)
